Keyboard Shortcuts:

  • B - Brush tool
  • E - Eraser
  • R - Rectangle
  • C - Circle
  • L - Line
  • T - Text tool
  • S - Selection tool
  • Ctrl+Z - Undo
  • Ctrl+Y - Redo
  • Ctrl+S - Save
  • Ctrl+O - Open
  • +/- - Zoom in/out
